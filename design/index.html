Design decisions
Key design tokens & choices

🎨 Brand colors

TruePoint Cobalt as accent for interactive elements, credit pill, and score highlights. Ink black (#111827) for primary buttons — calm, enterprise feel.

--accent: #2563C9

🖋️ Geist typeface

Geist for UI text, Geist Mono for code, API keys, masked contact data, score numbers, and keyboard shortcuts. Vercel-native font stack.

font: 'Geist', -apple-system

📐 Layout system

Sidebar #f9fafb · 1px hairline border · active nav #e8e8e8 fill (no colored indicator) · content #fff. Identical to Linear/Vercel monochrome system.

--sidebar-bg: #f9fafb

💳 Credits as pill, not nav

Credit balance lives in the topbar as a clickable pill that deep-links to Settings › Billing. Never a sidebar nav item — per IA spec.

topbar pill → Settings/Billing

🔒 Masked-first data

Email and phone show as ●●●●●●●● with a verification glyph (✓ valid · ⚠ risky · — none) until explicitly revealed. One-click Reveal opens cost confirmation.

reveal = 1 credit per field

📋 Panel-over-page nav

Import, record detail, builder, reveal, command palette — all open as drawers/panels over the persistent shell. Zero full-page navigations. Single-page command center.

Slide-over + ⌘K palette
