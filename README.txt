TRUEPOINT — BRAND KIT v1.0
==========================
"Aim true."  ·  The intelligent prospecting CRM.

The mark is three rising chevrons; the top "point" carries the brand
accent. The lead point earns the color — on light and dark.

COLOR — two-tone system
------------------------------------------------------------------
  Ink        #111827   text, mark lower chevrons, primary BUTTONS
  Surface    #FFFFFF / #F9FAFB
  Hairline   #F0F0F0
  Cobalt     #2563C9   BRAND ACCENT — fills & the mark only, NEVER text
  Cobalt 700 #1E4FA3   hover / darker fill
  Cobalt 50  #E9F0FC   soft tint background
  Apex tint  #5B8DEF   mark apex on dark surfaces

Rule: Cobalt is a FILL color, not a text color. All emphasis type and
buttons are Ink. Color signals brand; depth signals "do this."

USAGE RULES
------------------------------------------------------------------
DO   keep clear space around the mark equal to one chevron's height.
DO   reserve the accent (Cobalt) for the apex point and fills only.
DO   use the reversed files on dark surfaces.
DO   drop to the mono mark when color can't reproduce.
DON'T color the lower chevrons, or use Cobalt as a text color.
DON'T add a fourth chevron, rotate, outline, or point them downward.
DON'T set the wordmark in one uniform weight (True = 400, Point = 800).
